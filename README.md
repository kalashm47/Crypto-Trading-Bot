# Crypto Trading Bot 

A professional cryptocurrency trading bot prototype that connects to Binance public API, fetches real-time market data, calculates technical indicators, and generates trading signals based on moving average crossovers.

## Features

- 🔌 **Binance Integration**: Real-time price fetching from Binance public API
- 📊 **Technical Indicators**: SMA, EMA, and RSI calculations
- 📈 **Trading Strategies**: Moving average crossover strategy included
- 📝 **Comprehensive Logging**: Structured logging with log rotation
- ⚙️ **Configuration**: Environment-based configuration with .env support
- 🚀 **CLI Interface**: Easy-to-use command line interface
- 🧪 **Test Coverage**: Unit tests for indicator calculations
